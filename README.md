# LQA_ECF_2023

Prérequis :
- symfony
- symfony CLI
- php
- composer
- MAMP

Installation du projet en local :

- copier l'adresse du repo 
- ouvrir votre éditeur de code 
- aller dans l'onglet coloner
- copier l'adresse
- charger le code
- metre le dossier du projet dans le dossier htdocs de MAMP
- demarer MAMP
- ouvrire le terminal
- tapez la commande symfony serve
